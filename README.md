# TUT Bus Bot
TUT Bus Botは東京工科大学・八王子専門学校のスクールバス時刻を取得することができます。

## 使い方
### 友達追加
以下のQRコードを利用して `TUT Bus Bot` のアカウントを追加してください。

### バス時刻表取得
トーク画面を開くと以下の様なメニューが表示されるので取得したいバス種別を選択してください。
タップすると直近5本のバス発車時刻を取得することができます。
